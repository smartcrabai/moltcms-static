# @moltcms/static-json

Git-friendly JSON projection and transactional sync adapter for moltcms static sites.

## Install

```sh
npm install @moltcms/static-json
```

This package is part of [moltcms-static](https://github.com/smartcrabai/moltcms-static). See the repository README for setup, adapters, and examples.
