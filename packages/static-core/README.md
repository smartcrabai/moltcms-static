# @moltcms/static-core

Incremental route planning, dependencies, queries, and immutable generation core for moltcms static sites.

## Install

```sh
npm install @moltcms/static-core
```

This package is part of [moltcms-static](https://github.com/smartcrabai/moltcms-static). See the repository README for setup, adapters, and examples.
