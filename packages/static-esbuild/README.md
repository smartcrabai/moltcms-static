# @moltcms/static-esbuild

esbuild browser asset and server-module graph adapter for moltcms static sites.

## Install

```sh
npm install @moltcms/static-esbuild
```

This package is part of [moltcms-static](https://github.com/smartcrabai/moltcms-static). See the repository README for setup, adapters, and examples.
